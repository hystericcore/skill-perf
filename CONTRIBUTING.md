# Contributing to skill-perf

Thank you for your interest in contributing!

## Development Setup

```bash
git clone https://github.com/your-org/skill-perf.git
cd skill-perf
pip install -e ".[dev]"
```

## Running Tests

```bash
pytest
```

## Linting

```bash
ruff check src/
```

## Code Style

- Python 3.10+
- Type hints required
- Follow existing patterns in the codebase
